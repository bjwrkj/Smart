# Smart L
##[项目介绍]()
Smart L - 智能互联为家庭智能设备管理器，设计初衷主要为解决繁琐的设备管理，与硬件联动，采用实时侦测模式来及时发现和管理您的智能设备。此为Smart L的管理系统，还有与底层硬件交互的驱动程序，详情请戳：[iot-utils]("https://github.com/bjwrkj/iot-utils")
